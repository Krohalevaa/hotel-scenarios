Личный кабинет для генерации видео-сценариев

Авторизация через Supabase, хранение сценариев в PostgreSQL, управление профилем, аватаркой и историей генераций в одном интерфейсе.

🗂 История всех сценариев пользователя со статусами

Supabase Auth

Вход и регистрация

После успешной авторизации пользователь автоматически перенаправляется в кабинет.

Публичная конфигурация Supabase теперь загружается с сервера через /api/public-config. Диагностические сообщения пишутся в DevTools Console.
